# SoundController App 0.1.8

- Prevents multiple tray app instances from running in the same Windows user session.
- Adds MSI-based app update flow through `SoundController.Updater.exe`.
- Publishes both versioned MSI and setup bootstrapper installer files.
- Keeps GitHub as the default update feed location.
- Shows detected products in the update window even when they are already up to date.
